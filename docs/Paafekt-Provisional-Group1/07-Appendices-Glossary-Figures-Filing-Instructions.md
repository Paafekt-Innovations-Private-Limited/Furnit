## APPENDIX A — GLOSSARY OF TECHNICAL TERMS

**Affine-invariant depth map:** A depth map in which depth values are correct up to an unknown multiplicative scale factor and additive offset, representing relative depth ordering and proportional depth differences without absolute metric values.

**Bundle adjustment:** A nonlinear least-squares optimization procedure that jointly refines estimates of camera poses and three-dimensional feature point positions to minimize the sum of squared reprojection errors across all observations.

**Depthwise-separable convolution:** A factored convolutional neural network layer that applies a separate spatial filter per input channel followed by a 1×1 convolution across channels, substantially reducing computation relative to standard convolution while preserving representational capacity.

**Environment map:** A representation of the incident illumination at a point in space from all directions, typically stored as a spherical or cube-map image.

**Five-point algorithm:** A minimal solver for estimating the relative pose (rotation and translation up to scale) between two camera views from five point correspondences.

**Heterogeneous computing:** The use of multiple distinct types of processing units (e.g., CPU, GPU, NPU) within a single computing system, each assigned tasks suited to their computational characteristics.

**Knowledge distillation:** A model compression technique in which a smaller "student" neural network is trained to mimic the outputs of a larger "teacher" network.

**LiDAR:** Light Detection and Ranging; a depth sensing technology that measures distances by emitting laser pulses and measuring the time of return, used in premium mobile devices for precise depth measurement.

**Markov random field (MRF):** A probabilistic graphical model representing joint distributions over spatially-structured variables, used for spatially-regularized image segmentation and labeling.

**Monocular camera:** A single-aperture image sensor producing standard two-dimensional photographic images without depth information.

**Neural processing unit (NPU):** Dedicated hardware accelerator in modern mobile devices optimized for efficient execution of neural network inference operations.

**Physically-based rendering (PBR):** A rendering methodology that models light interaction with surfaces using physically-accurate bidirectional reflectance distribution functions (BRDFs).

**Post-training quantization:** The process of reducing the numerical precision of trained neural network weights after training is complete, without further training.

**Sparse voxel representation:** A data structure that represents a volumetric grid by storing data only for occupied voxels, using a hash map indexed by voxel coordinates, as opposed to a dense 3D array.

**Visual odometry:** The process of estimating camera motion from image sequences by tracking visual features across frames.

**Voxel:** A three-dimensional generalization of a pixel; a unit of volume in a three-dimensional grid.

## APPENDIX C — FILING INSTRUCTIONS FOR SELF-FILER

**Step-by-step USPTO provisional patent filing via Patent Center (www.patentcenter.uspto.gov):**

1. **Create a USPTO account** at account.uspto.gov if not already registered.
2. **Log in to Patent Center** at patentcenter.uspto.gov.
3. **Start a new submission:** Select "File a New Application" → "Provisional Application."
4. **Prepare Application Data Sheet (ADS):** Download Form PTO/SB/14 from USPTO.gov. Complete:
   - Inventor: Kishore Shivanna, [address]
   - Applicant: Paafekt Inc., [address], Delaware corporation
   - Title: [Title of the specific patent being filed]
   - Correspondence address: [Your address]
5. **Prepare Micro Entity Certification:** Download Form PTO/SB/15A. Complete and sign. Certify that the applicant qualifies as a micro entity based on gross income basis.
6. **Upload documents:** Upload this specification (as PDF), the ADS, and the Micro Entity Certification.
7. **Pay filing fee:** $320 per provisional application (micro entity rate). Payment by credit card through Patent Center.
8. **File for EACH of the 6 patents separately.** Total fees: 6 × $320 = $1,920.
9. **Save your filing receipts:** USPTO will issue a Filing Receipt with an Application Number and Filing Date for each provisional. Save these receipts — the filing date on the receipt is your priority date.
10. **Calendar the 12-month deadline:** You must file a corresponding non-provisional patent application within 12 months of each provisional's filing date to claim priority benefit. Mark these deadlines immediately.

**Important:** Do not use any specific title from the FurnitureFit codename. Always use "Paafekt" in all application materials.
